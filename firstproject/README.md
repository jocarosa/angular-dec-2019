# POKEMON API

This project consumes a pokemon API retrieving all Pokemons by Generation.

https://pokeapi.co/

# TECH:

-Angular 8
-Typescript 3.5
-Rxjs 6.4 ( folkJoin, switchMap )


# Best manner:

-Lazy load
-Single Responsability principle
-Naming conventions
-Breaking down Components
-url alias
